SCOUT User Manual

Introduction

SCOUT is a powerful tool that allows you to upload data in CSV format, cluster the data according to individual distance information, and visualize the results.

Getting Started

Step 1: Data Submiting:

Scout offers data analysis and visualization for .json files. You can input data directly into a text box or upload files via a button. Scout supports CSV files or manually entered data in CSV format. For JSON files, it will automatically generate cluster diagrams.

  1. Locate the section titled Upload your CSV file at the top left of the SCOUT interface.
  2. Click on the ADD DATA and a dialog box will open.
  3. Navigate to the location of your CSV file on your computer.
  4. Select the file and click `Open` to upload it into the SCOUT system.

Clustering Data

Step 2: Data Preprocessing:

  1. Locate the section titeled Choose clustering , click on the CHOOSE YOUR DATA TYPE for data configuration.
  2. Select the data type you're submitting and choose the corresponding distance equation to better visualize data structures. Custom allows a self-written distance function in JavaScript. Preclustered lets you upload an already processed distance matrix.
  3. The SCALING ALGORITHM affects cluster division preferences and methods, while DISTANCE CALCULATION distance calculation determines how distances are computed.

Table interpretation

Step 3: Flag match

  1. Locate the bottom section titled Flag Match .
  2. Use the FLAG CHECK to match flags for each columns in your data.
  3. Repeatedly clicking the FLAG CHECK button will reset the match list. While the DELETE ALL button clears all inputs and selections, including data and custom options.
  4. Click here to learn more about flags.
  • For multi-column vector data, marking all distance parameters with the 'distance information' flag is crucial.
  • Additional flags for detailed within-cluster distribution and weight analysis are optional.
  • Column names are auto-detected from your data, but manual selection is also possible. Ensure column names match tags correctly; the order does not affect analysis outcomes.

The map

  1. Click RUN to obtain analysis results. For large datasets, processing may take 10-30 seconds . Please do not refresh the page during this time.
  2. The initial map is usually positioned in the top left corner of the display area with a large zoom level. You can drag the image by holding the left mouse button and adjust the zoom with the scroll wheel.
  3. The + and - bottons in the bottom right corner of the display area represent the current cluster level (1-20), with higher levels offering more detailed segmentation.
  4. Clicking on a circle representing a cluster allows you to view current cluster information on the left side of the page. Basic information includes the level and number of members. If additional information was selected in flag match, detailed analysis will be presented.
  5. Clicking FULLSCREEN enters fullscreen mode, which can be exited with the ESC key.
  6. Export will download a json file containing the map structure and analysis information.
  7. Restart resets all input information, and you will not be able to return to view previous results.

SCOUT is a powerful tool, but the quality of the results depends heavily on the appropriateness of the distance measure and visualization type for your specific dataset. Always ensure your choices align with your data and the insights you hope to gain.

Enjoy exploring your data with SCOUT!
